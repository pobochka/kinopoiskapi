SPA-приложение для просмотра информации о фильмах на базе [kinopoisk.dev](https://kinopoisk.dev/).

### 1. Установить зависимости
npm install

### 2. Настроить переменные окружения
cp .env
# API-ключ в .env

### 3. Запустить
npm run dev

VITE_KINOPOISK_API_KEY = API-ключ с kinopoisk.dev 
VITE_KINOPOISK_API_URL = https://api.kinopoisk.dev/v1.4
